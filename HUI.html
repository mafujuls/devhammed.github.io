#. Introduction to HUI.js

HUI.js is a Simple and Lightweight JavaScript UI Layout Library for creating easy and re-usable Webpage Layouts. You can download it locally HERE or link to it using SCRIPT tag:-

<script src="http://www.oyedelehammed.ml/HUI.js"></script>

#. Getting Started

There are 2 main functions in HUI Object, namely:-

HUI.Create(attributes, text, element): This would create an element as a parent or container, the "text" argument will contains HTML/Text and its optional, "attributes" argument is a JavaScript Object that contains the element attributes and its optional too, "element" attribute is the HTML Element to create and its optional too (HUI.js creates SECTION element by default). HUI.Create() would return two function, thats parentElement.Child(childElement, text, attributes) which can be used to add Child element to the Parent element and parentElementTwo.Join(parentEkementTwo) which can be used to merge/join Parent elements. ( More explanation below ).

HUI.Render(element, viewID) : This would render/display the element created by HUI.Create() on the view/element with the ID attribute passed.

#. Example and Demo

Below is an example code that includes all functions in HUI.js, First make sure you have include the "HUI.js" file in your page footer, then followed by your SCRIPT tag code or JavaScript file include e.g:

<div id="hui-view"></div>
<script src="HUI.js"></script>
<script>
var header = HUI.Create( {'id': 'header'}, '<h1>Header Title</h1>', 'header' );
header.Child( 'nav', 'Navigation', {'id': 'nav'} );
var main = HUI.Create( {'id': 'main'}, '<h2>Main content</h2>', 'main' );
main.Child( 'p', "Hello world" );
var footer = HUI.Create( {'id': 'footer'}, "Footer Text", 'footer' );
var container = HUI.Create( {'id': 'container'} );
container.Join( header ).Join( main ).Join( footer );
HUI.Render( container, 'hui-view' );
</script>
